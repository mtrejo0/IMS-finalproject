wavegen.py¶

class imslib.wavegen.SpeedModulator(generator, speed=1.0)¶

Modulates the speed of generated data from a source.

Parameters

generator – The generator object. Must define the method generate(num_frames, num_channels), which returns a tuple (signal, continue_flag).

generate(num_frames, num_channels)¶

Generates output of modulated speed by resampling audio data according to the specified speed.

Parameters

  • num_frames – An integer number of frames to generate.

  • num_channels – Number of channels. Can be 1 (mono) or 2 (stereo)

Returns

A tuple (output, True). The output is the audio data from wave source, a numpy array of size num_frames * num_channels.

set_speed(speed)¶

Sets the factor by which the speed should be modulated. For example, a speed of 1.0 is the original speed, 2.0 is twice as fast, 0.5 is twice as slow.

Parameters

speed – The desired speed, a float.

class imslib.wavegen.WaveGenerator(wave_source, loop=False)¶

Generates audio data by asking an audio-source (eg, WaveFile) for that data.

Parameters

  • wave_source – The source of data. Must define get_frames(start_frame, end_frame), which returns a numpy array.

  • loop – When True, continuously restarts playback from beginning of wave source.

generate(num_frames, num_channels)¶

Generates output from the wave source. When paused, only zeros are generated. When looping, if the end of the buffer is reached, more data will be read from the beginning.

Parameters

  • num_frames – An integer number of frames to generate.

  • num_channels – Number of channels. Can be 1 (mono) or 2 (stereo)

Returns

A tuple (output, True). The output is the audio data from wave source, a numpy array of size num_frames * num_channels.

get_gain()¶
Returns

The volume/gain of the generator, a float between 0 and 1.

pause()¶

Pauses audio generation.

play()¶

Starts audio generation from the last frame it played.

play_toggle()¶

Toggles play and pause.

release()¶

Stops any further audio generation.

reset()¶

Restarts playback from frame 0.

set_gain(g)¶

Sets volume/gain value for audio output.

Parameters

g – A float specifying gain. Will be clipped between 0 and 1, where 1 is full volume.

imslib.wavegen.convert_channels(data, in_channels, out_channels)¶

Convert an audio data buffer of a given number of channels to a different number of channels. Currently only handles going from mono to multi-channel, or from multi-channel to mono.
